HomeController

HomeController class Null safety

A controller class that manages the Layouts. It extends GetxController class. To get an instance of this class, use:

final HomeController = Get.put(HomeController()); //creates a new controller
or
final HomeController = Get.find<HomeController>(); // finds an existing controller
