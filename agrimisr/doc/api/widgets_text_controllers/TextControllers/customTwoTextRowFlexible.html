A Text widget that displayes two texts, a tag and a body. It takes this form:

titleText (tag) : text (body), the body text Does not expand to fill the Row, instead it is wrapped in a Flexible widget. the tag is Bold.

Implementation

Widget customTwoTextRowFlexible({
  String? titleText,
  String? text,
  double? fontSize,
  int? maxLines,
  Color? textColor,
}) =>
    Padding(
      padding: MyPadding.hPadding,
      child: Row(
        crossAxisAlignment: CrossAxisAlignment.baseline,
        mainAxisSize: MainAxisSize.min,
        textBaseline: TextBaseline.alphabetic,
        children: [
          Text(
            '${titleText ?? ''}${titleText != null ? ': ' : ''}',
            maxLines: 1,
            overflow: TextOverflow.ellipsis,
            style: TextStyle(
              fontSize: fontSize ?? 14,
              fontWeight: FontWeight.bold,
              color: textColor,
            ),
          ),
          Flexible(
            fit: FlexFit.loose,
            child: Text(
              text ?? '',
              maxLines: maxLines ?? 1,
              overflow: TextOverflow.ellipsis,
              style: TextStyle(
                fontSize: fontSize ?? 14,
                color: textColor,
              ),
            ),
          ),
        ],
      ),
    );
